# Pixiv Gallery

Pixiv Gallery is a light web application built with JavaScript and Pug. It provides a platform for users to browse and interact with a variety of illustrations.

## Features

- **Dark Theme**: The application has a dark theme for a comfortable viewing experience.
- **Image Gallery**: Users can browse through a vast collection of illustrations.
- **Login Status**: The application provides a login feature and displays the login status of the user.
- **Navigation**: The application provides easy navigation to various pages such as Home, Login, Recommended Illustrations, Bookmarked Illustrations, and User Gallery.

## Getting Started

To get a local copy up and running, follow these simple steps:

1. Clone the repository
```bash
git clone <repository-link>
```
2. Install NPM packages
```bash
npm install
```
3. Start the server
```bash
npm start
```
4. Open your browser and navigate to `localhost:3000`

## Demo

You can check out the online demo of the application at [https://pixivimageviewer.fly.dev/](https://pixivimageviewer.fly.dev/). Please note that the online version might be unstable due to limited resources.
